# 🔐 Password Generator (GUI)

A simple GUI-based password generator built using Python and Tkinter.

## 🚀 Features

* Generate strong passwords
* Custom length
* Include uppercase, digits, and symbols
* Copy to clipboard

## 🛠️ Tech Used

* Python
* Tkinter
